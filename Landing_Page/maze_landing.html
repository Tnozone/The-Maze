The Maze

The Maze is an educational project of making a game with raycasting.

  • Feature
  • About

Feature

Game in a 3D environment.

The Maze is a small game where you can move around, open vine walls after picking up a knife, and die to a monster.

About

This project was inspired by the game DOOM. I wanted to learn to make a game like it through coding, and this project was a step towards that.
